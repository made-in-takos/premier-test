Selecteurs CSS vs SCSS

#main {
  color: #999;
}
#main .content {
  color: #bfbfbf;
}

L'extrait CSS ci-dessus peut être écrit dans SCSS comme suit: 


#main{
  color: #999;
  .content{
    color: #bfbfbf;
  }
}

As we can see SCSS strucuture is more like actual DOM structure.This allows us to organise our CSS more effectively

Examples:


#main h1 {
  color: #00ff00;
  width: 97%;
  .title {
    background-color: #ff0000;
    color: #000000;
  }
}

Will be compiled to :


#main h1 {
  color: #00ff00;
  width: 97%;
}
#main h1 .title{
  background-color: #ff0000;
  color: #000000;
}
Referencing parent selector:

#wrapper h1{
  width:20px;
  &:hover{
    width:25px;
  }
}

& Will be replaced with parent selector.The above code is equivalent to


#wrapper h1{
  width:20px;
}
#wrapper h1:hover{
  width:25px;
}
