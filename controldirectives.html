DIRECTIVES DE CONTRÔLE


Il est à noter que SASS reconnait les boucles $for, $each , $while ainsi que les conditions @if, @else et @else if.

La condition @if

Comme dans tous language de programmation, pour qu'une condition soit remplie, celle-ci doit est égale à TRUE. Dans ce cas ci, si 1 + 1 == 2 alors ..., sinon si 5 est inférieur à 3 alors ..., sinon NULL

1 + 1 est bien égale à 2 (ce qui renvoie TRUE), alors que 5 n'est pas inférieur à 3 (ce qui renvoie FALSE) et NULL renvoie FALSE.
Nous voyons que seul la première condition est égale à TRUE, ce sera donc une bordure de 1px solid 


p {
 @if 1 + 1 == 2 { border: 1px solid;  }
 @else if 5 < 3      { border: 2px dotted; }
 @else null       { border: 3px double; }
}

Ce qui sera compilé en:


p {
 border: 1px solid;
}

Les boucles

Il est aussi interessant de pouvoir créer différents types de boucles, de manière à automatiser des tâches.

@for

La boucle @for permet l'incrémentation d'une variable, (dans ce cas ci $i) jusqu' à un nombre donné (1200px).
Où, 3 est le nombre de colonnes et 1200px, la taille de l'écran, par exemple.

ce qui est calculé comme suit :
1200 / 3 = 400.

Nous avons donc trois colonnes d'une l'argeur de 400px chacune. 


$nbrCol : 3 ;
$largCol : 1200px / $nbrCol ;

@for $i from 1 to ( $nbrCol + 1 ) {
 .col-#{$i} {
   width: ($largCol * $i) ;
  }
}

Ce qui sera compilé en:


.col-1 {
  width: 400px;
}
.col-2 {
  width: 400px;
}
.col-3 {
  width: 400px;
}

@each

Cette boucle permet de parcourir, à l'aide d'une boucle, tous les éléments d'un tableau (ici $header).
Cette boucle peut saverer très pratique dans le cas d'une gallerie d'images par exemple, en effet, celle-ci permet d'automatiser le chargement sans devoir chager les images une à une. des images. 


@each $images in 01, 02, 03, 04 {
  .#{$images}-thumb {
    background-image: url('../images/#{$images}.png');
  }
}

Ce qui sera compilé en:


.01-thumb {
  background-image: url('../images/01.png');
}
.02-thumb {
  background-image: url('../images/02.png');
}
.03-thumb {
  background-image: url('../images/03.png');
}
.04-thumb {
  background-image: url('../images/04.png');
}

@while

Avec la boucle while, tant qu'il reste des valeurs, la condition est remplie ( == TRUE ), et donc, la tâche se poursuit.

Dans ce cas ci, tant que 6 n'est pas inférieur à 0 alors $i - 2 


$i: 6;
@while $i > 0 {
 .item-#{$i} {
   width: 2em * $i;
 }
$i: $i - 2;
}

Ce qui sera compilé en:


.item-6 {
  width: 12em;
}

.item-4 {
  width: 8em;
}

.item-2 {
  width: 4em;
}
