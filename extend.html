@EXTEND


L'utilisation de @extend nous permet de partager un ensemble de propriétés CSS d'un sélecteur à un autre.

Plutôt que d'encombrer notre code HTML de classes supplémentaires pour gérer de petites exeptions, nous pouvons utiliser la fonction @extend de SASS pour "enchaîner" les styles partagés par plusieurs sélecteurs.
Il nous est alors possible d'ajouter des règles supplémentaires pour créer un style unique, sans avoir à dupliquer les les styles partagés.

Ainsi, si nous voulons utiliser @extend pour gérer différents types de bouttons, nous pouvons simplifier le code HTML en n'incluant qu'une seule classe. 

<button class="btn">Normal</button>
<button class="btn-blue">Blue</button>
Les deux boutons auront les mêmes styles, sauf la couleur du background.

.btn{
  margin:10px;
  color:black;
  width:200px;
  height:100px;
}
.btn-blue{
  @extend .btn;
  background:blue;
}
Ce qui sera compilé en:

.btn,.btn-blue{
  margin:10px;
  color:black;
  width:200px;
  height:100px;
}
.btn-blue{
  background:blue;
}

Nous pouvons également inclure et utiliser plusieurs classes dans une déclaration à l'aide de dette fonction. 


.btn{
  margin: 10px;
  color: black;
  width: 200px;
  height: 100px;
}
.font-color{
  color: white;
}
.btn-blue{
  @extend .btn;
  @extend .font-color;
  background: blue;
}
Ce qui sera compilé en:

.btn,.btn-blue{
  margin: 10px;
  color: black;
  width: 200px;
  height: 100px;
}
.btn-blue{
  color: white;
  background: blue;
}
Le placeholder %

Les placeholders nous permettent de déffinir des classes "fantomes" qui n'apparaîtront pas dans le CSS final en tant que telles.

Pour préciser qu'il sagit d'un placeholder, nous allons préfixer le nom de la class par le symbole % au lieu du point.

Nous pourrons ensuite appeler ces règles dans d'autres classes comme nous l'avons fait précédemment, à l'aide de la fonction @extend. 


%btn{
  margin:10px;
  color:black;
  width:200px;
  height:100px;
}
.btn-blue{
  @extend %btn;
  background:blue;
}
.btn-red{
  @extend %btn;
  background:red;
}

Ce qui sera compilé en:


.btn-blue{
  margin:10px;
  color:black;
  width:200px;
  height:100px;
  background:blue;
}
.btn-red{
  margin:10px;
  color:black;
  width:200px;
  height:100px;
  background:red;
}

@extend VS @mixins

Alors qu'un @mixin recopiera la règles dans chaque déclaration, @extent créera des selecteurs multiples séparés par les virgules pour les styles partagés.
