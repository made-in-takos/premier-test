VARIABLES


Les variables SCSS, comme dans d'autres languages de programmation, sont des éléments auquels on donne un nom et une valeur qui, comme son nom l'indique, varie. Celles-ci peuvent etre des valeurs numériques ou exadécimales, des liens, des fonts, etc...

En SCSS, on definie une variable avec le symbole $.
une fois définies, elles peuvent être invoquées dans des déclarations. 


$width: 10px;
$headercolor:red;

#main h1{
  width: $width;
  color:$headercolor;
}

Ce qui sera compilé en:


#main h1{
  width: 10px;
  color:red;
}

Grace à ces variables, il est beaucoup plus rapide et faciles de modifier des couleurs ou des tailles, puisque une seule modification apportée à une variable, sufit à modifier les valeurs dont le nom est répétés dans la feuille de style.

Cependant, les variables ne sont disponibles que dans le niveau des sélections imbriquées où elles sont définies. Si vous avez besoin de cette variable en dehors de cette portée, vous devez ajouter !global. 


#main {
  $width: 5em !global;
  width: $width;
}
#sidebar {
  width: $width;
}

Vous pouvez spécifier des variables dans le fichier sperate et vous pouvez importer ce fichier scss Pour utiliser ces variables. Il agit comme fichier de configuration pour votre projet. Si vous changez la valeur de la variable, elle sera modifiée partout où elle est utilisée. Il est très efficace dans les grands projets. Nous le verrons plus tard.

Vous pouvez également effectuer des opérations dans SCSS.

Par exemple: 


p {
//CSS simple, pas de division
  font: 10px/8px;
  $width: 1000px;
//Utilise une variable, la division
  width: $width/2;
//Utilise une fonction, la division
  width: round(1.5)/2;
//Utilise les parenthèses, la division
  height: (500px/2);
//Utilise +, la division
  margin-left: 5px + 8px/2px;
//Dans une liste, les parenthèses ne comptent pas
  font: (italic bold 10px/8px);
}

Ce qui sera compilé en:


p {
  font: 10px/8px;
  width: 500px;
  height: 250px;
  margin-left: 9px;
}

Interpolation: #{}

Vous pouvez utiliser des variables dans les noms de propriétés ainsi que les valeurs, en utilisant # {}. 


$name: foo;
$attr: border;
p.#{$name} {
  #{$attr}-color: blue;
}

Ce qui sera compilé en:


p.foo {
  border-color: blue;
}
