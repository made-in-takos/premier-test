FUNCTIONS


Vous pouvez également définir des fonctions qui renverons une valeur.

une fonction comporte une séquence d'instructions réalisant un calcul ou une tâche. celles-ci sont très similaire aux un mixins, mais la sortie d'une fonction est une valeur unique. Cela peut être n'importe quel type de données SCSS, y compris:
  • Des nombres
  • Des chaînes
  • Des couleurs
  • Des booléens
  • Des listes
Par exemple:

$no-of-columns:12;
$gutter-width:2%;

@function get-column-width($i){
  @return (100%
	- ($no-of-columns/$i - 1)
	* $gutter-width)
	/ $no-of-columns * $i;
}

.col-6{
  width:get-column-width(6);
}
Ce qui sera compilé en:

.col-6 {
  width: 49%;
}
une Fonction à donc pour avantage de définir un ensemble de propriétés au seins d'un même bloc et de pouvoir le réutiliser en l'appelant dans une déclaration.
